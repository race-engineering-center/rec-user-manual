# Библиотека кругов

Библиотека кругов — удобный способ организовать телеметрию и быстро находить нужные
круги.

![Окно библиотеки](img/analyzer_lap_library_src.png "Окно библиотеки")

Слева отображаются все трассы, машины и пилоты из локальной библиотеки. Снимите галочку
с любого элемента, чтобы скрыть ненужные круги. Это не удаляет их из библиотеки, а
просто скрывает круги из основного окна.

![Фильтры](img/analyzer_lap_library_filters_src.png "Фильтры")

Как и в дереве кругов основного окна, можно удерживать Alt (Windows) или Option (macOS),
чтобы показать только выбранный элемент и скрыть остальные.

![Эксклюзивный выбор](img/analyzer_library_tree_selection.gif "Эксклюзивный выбор")

!!! note

    Настройки фильтров сохраняются между запусками Analyzer.

Также можно фильтровать круги через поле поиска в верхней части окна.

![Поиск](img/analyzer_lap_library_search_src.png "Поиск")

Введите запрос — таблица обновится и покажет только строки, содержащие все указанные
слова. Фильтр работает по принципу "И": строка отображается только если содержит все
слова из запроса.

Оба способа фильтрации можно комбинировать.

## Загрузка кругов для анализа

Чтобы загрузить круги для анализа, выберите их в окне библиотеки и перетащите в список
кругов. Подробнее — в разделе [Lap info list](laps.ru.md).
